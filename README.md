# tokokita

Hardened Node.js e-commerce — helmet, CSRF, rate-limit, atomic stock management.

## Stack

- Python 3.11+
- PyTorch / scikit-learn
- See `requirements.txt` or `pyproject.toml`

## Quick Start

```bash
git clone https://github.com/afuckingco/tokokita.git
cd tokokita
pip install -r requirements.txt
```

## CI

GitHub Actions — lint (ruff), build (py_compile). Every push triggers a run.
